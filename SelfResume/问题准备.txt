1. React相关

a. 生命周期相关
  组件被实例化，getInitialState => componentWillMount => render => componentDidMount

  每次state发生更改，componentWillReceiveProps => shouldComponentUpdate => componentWillUpdate => render => componentDidUpdate


render方法会创建一个虚拟DOM树，这也是React高效的原因
组件的props属性可以通过父组件来更改，这个时候会调用componentWillReceiveProps，可以在这里设置state的值
shouldComponentUpdate(nextProps, nextState),返回false的话，后面的WillUpdate等不会执行(这里不允许在更新props或state)
